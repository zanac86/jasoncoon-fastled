# Lite version without web server

* Auto change palettes and patterns with random order
* Can work on `ESP8266` and `Arduino Pro Micro`

| Button clicks | Mode               |
|---------------|--------------------|
| 1             | Next pattern       |
| 2             | Change brightness  |
| 3             | Random solid color |
